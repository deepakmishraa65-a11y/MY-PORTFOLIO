# 🎨 My Portfolio — Deepak Mishra

A modern, animated portfolio website for a **Designer / Creative**, built with **React + Vite + Tailwind CSS** and **Framer Motion** animations.

## ✨ Features

- ⚡ Fast — powered by Vite
- 🌗 Dark / Light mode (auto-detects system, remembers your choice)
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎬 Smooth scroll animations (Framer Motion)
- 🖼️ Filterable project gallery
- 📊 Animated skill bars
- 🎯 Sections: Hero, About, Services, Skills, Work, Contact

## 🚀 Getting Started

```bash
# 1. Install dependencies
npm install

# 2. Start the dev server
npm run dev

# 3. Build for production
npm run build
```

Open the printed local URL (usually http://localhost:5173) in your browser.

## ✏️ How to Edit Your Content

All your content lives in **one file** — easy to update:

```
src/data.js
```

Edit it to change:
- Your **name, role, tagline, email** (`profile`)
- **About** text and stats (`about`)
- **Skills** and levels (`skills`)
- **Projects** — title, category, image, link (`projects`)
- **Services** you offer (`services`)
- Social links — Instagram, Behance, Dribbble, LinkedIn, GitHub

> 💡 Tip: To use your own project images, put them in the `public/` folder
> and set the image path like `/my-project.jpg`, or paste any image URL.

## 🌐 Deploy (free)

- **Vercel** — import this repo at [vercel.com](https://vercel.com) → auto-deploys.
- **Netlify** — drag the `dist/` folder, or connect the repo.
- **GitHub Pages** — run `npm run build` and serve the `dist/` folder.
